###时间复杂度
```java
int num1, num2;

for(int i=0; i<n; i++){ 
	num1 += 1;

	for(int j=1; j<=n; j*=2){ 

		num2 += num1;

	}
}
```
首先，我们来看外循环for（i=0；i<n；i++），按照i++的递加速度，直到这个循环退出，一共是n次。再来看看内部循环，for（j=1；j<n;j*=2）,这个内部循环的累加速度是j=j*2，假设循环x次之后，这个循环退出，2^x>n===》x=log(2)N ps:(2)代表底数，请原谅我打不出正确的数学符号，sorry如果把两个循环合在一起看的话，也就是一共循环了n个x次，也就是n*log(2)N

   当n→∞时，(1+1/n)^n的极限
